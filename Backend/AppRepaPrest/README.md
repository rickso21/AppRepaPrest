# 🚀 AppRepaPrest - Sistema de Préstamos y Ventas

Sistema de monitoreo de préstamos, ventas y gestión financiera desarrollado con **Laravel 12** y **React 19.1.0**.

---

## 📋 Requisitos del Sistema

Antes de comenzar, asegúrate de tener instalado:

| Herramienta | Versión Mínima | Verificar |
|------------|---------------|-----------|
| **PHP** | 8.2 o superior | `php --version` |
| **Composer** | 2.x | `composer --version` |
| **Node.js** | 20.x o superior | `node --version` |
| **NPM** | 10.x o superior | `npm --version` |
| **MySQL/TiDB** | 5.7 o superior | - |
